# Skill_Factory_Homework_Module_18

1. Программа при запуске создает файлы "users.txt" и "messages.txt", куда записывает состояния объектов соответствующих классов, 
2. Устанавливает права доступа на чтение и запись только для пользователя, который запускает программу,
3. Считывает статусы пользователей и сообщений из сформированных файлов и выводит в консоль.
